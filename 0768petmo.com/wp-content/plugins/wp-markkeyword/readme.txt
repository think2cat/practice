== Description ==

This plugin can high light keyword after someone searchecd it, which supports the search engines like google, baidu, ask, yahoo&bing.<br />
1.the highlight words are from the link of the search engine<br />
2.every key word is added with a link, clicking the link can search in the site<br />
3.on the top of the article show the key words which users are searching<br />
4.high light the searching result<br />
5.the highlight words are showed in the content and the excerpt<br />
