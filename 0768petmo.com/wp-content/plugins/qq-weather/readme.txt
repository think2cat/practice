== Description ==
This pulgin generates qq weather for WordPress Blog. data from qq.com
以QQ首页天气数据为基础，支持国内所有城市, 根据访客IP自动获取所在城市的天气。
只需指定需要显示页面ID，支持多粒，自定义模板、样式、png图片透明。
异步加载, 代码压缩，缓存JS, 对网站的性能影响较小。
简单容易上手，可设置开启和关闭天气预报

== Installation ==

1. Upload `baidu-tracker.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. admin -> "Settings" -> "QQ weather"

1、下载此插件并上传到wp-content\plugins目录中；
2、登录网站后台激活此插件；
3、然后进入“设置”，“QQ weather” 进行设置。


在页面或者模板中定义一个需要显示天气的容器ID，例如：
<p id="WeatherContainerId"></p>
然后填写页面ID容器，点击保存， 此时将在指定ID显示天气，类似本站右上角天气显示。
